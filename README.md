# Sistema de Gerenciamento de Bar (Algoritmos II)

Sistema desenvolvido em C++ para controle de estoque, compra e venda de bebidas. Este projeto foi desenvolvido como parte da disciplina de Algoritmos II na UNICENTRO.

## 🚀 Funcionalidades
- Cadastro e edição de produtos.
- Registro de entradas (compras) e saídas (vendas).
- Gerenciamento de estoque em tempo real.

## 🛠️ Tecnologias e Ferramentas
- **Linguagem:** C++
- **Compilador:** GCC/G++
- **Ambiente:** Terminal/CLI
- **Containerização:** Docker

## 🐳 Rodando com Docker

Se você tem o Docker instalado, pode rodar este projeto sem precisar configurar o compilador localmente:

1. Construa a imagem:
   `docker build -t sistema-bar .`

2. Execute o container:
   `docker run -it sistema-bar`
